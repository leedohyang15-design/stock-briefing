# 📈 개인 투자자 맞춤형 주식 브리핑 자동화 시스템 개발 요구사항

## 1. 프로젝트 개요
- **목표:** 개인 투자자(개미 투자자)가 복잡한 시장 상황을 출근길에 1분 만에 파악할 수 있도록, 핵심 정보만 요약된 데일리 주식 브리핑을 매일 아침 자동으로 발송하는 프로그램 개발.
- **실행 주기:** 매 거래일(월~금) 오전 9:00 KST (공휴일 제외)
- **발송 채널:** 텔레그램 봇, 슬랙, 또는 이메일 (개발 용이한 채널로 우선 설정 요망)
- **사용 언어:** Python 추천 (요청 시 변경 가능)

## 2. 브리핑에 포함될 핵심 섹션 (출력 포맷)
프로그램은 웹 크롤링 및 API를 통해 데이터를 수집하고, LLM(Claude API 등)을 활용해 아래 4가지 섹션으로 요약하여 메시지를 구성해야 합니다.

### 📊 [섹션 1] 간밤의 글로벌 증시 & 오늘의 관전 포인트
- **내용:** 미국 주요 지수(다우, 나스닥, S&P500) 마감 수치 및 1~2줄 시황 요약.
- **핵심:** "그래서 오늘 국내장은 어떤 섹터가 유리한가?"에 대한 짧은 인사이트 제공.

### 🔥 [섹션 2] 전일 시간외 특징주 & 급등 종목 (이유 필수)
- **내용:** 전일 정규장 및 시간외 거래에서 급등/급락한 특징주 상위 3~5개.
- **핵심 포맷:** 단순히 종목명만 나열하지 않고, **`[종목명] : 상승 이유 1줄 요약 (예: 엔비디아 호실적에 따른 HBM 수혜 기대감)`** 형태로 반드시 이유를 매칭할 것.

### 🚨 [섹션 3] 오늘의 주요 일정 & 리스크 알리미
- **내용:** 개인 투자자가 물리거나 뇌동매매하지 않도록 주의해야 할 요소를 안내.
- **포함 사항:** - 대규모 보호예수(락업) 해제 종목
  - 주요 거시 경제 지표 발표 일정 (예: CPI 발표일)
  - 단기 과열로 인한 거래 정지 예고 종목

### 💡 [섹션 4] 최신 증권사 리포트 요약 (선택적)
- **내용:** 당일 아침 새로 발간된 리포트 중 '목표가 상향' 또는 '신규 테마'와 관련된 리포트 2~3개 요약.

## 3. 기술적 요구사항 및 지시사항 (Action Items for Claude)
1. **데이터 수집 방법 제안:** 위 섹션들을 구현하기 위해 어떤 API(예: Yahoo Finance, 한국투자증권 OpenAPI, OpenDart, 네이버 금융 크롤링 등)를 사용하는 것이 가장 안정적인지 구조를 설계해 주세요.
2. **LLM 요약 로직 추가:** 크롤링한 날것의 기사나 데이터를 클로드 API 등을 이용해 '개인 투자자가 읽기 쉽게 1줄 요약'하는 프롬프트 체인을 코드 내에 포함해 주세요.
3. **스케줄링 구현:** 매 거래일 9시에 정확히 실행되도록 `schedule` 라이브러리, Github Actions, 또는 AWS EventBridge 등을 활용한 자동화 배포 방법과 코드를 작성해 주세요. (한국의 공휴일/휴장일은 건너뛰는 로직 포함)
4. **결과물:** 전체 Python 코드와 `requirements.txt`, 그리고 실행을 위한 리드미(Setup Guide)를 단계별로 작성해 주세요.
